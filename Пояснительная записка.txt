Проект содержит сайт на Flaske с различным функционалом созданным при помощи vk api.
Функционал:
Поиск связей между двумя страницами по теории 6-ти рукопожатий
Анализ друзей(полностью рекоднуто, по сравнению с первым проектом работает моментально)
Получение ID страницы

Файлы:
main.py подгрузка шаблонов основной функционал
use_db.py работа с базами данных
vki.py анализ друзей и получение ID
rk_get.py 6-ть рукопожатий, две версии для обычных людей и людей со своим токеном
templates 21 шаблон html страниц

Примечание: Бустрап использовался но не везде, например он есть в шаблоне change_password

Ошибки в 6-ти рукопожатиях: vk не нравится рекурсия такого уровня, и такое кол-во запросов в секунду. Поэтому он может иногда кидать ошибки, однако после перезапуска поиска все должно быть ок.
